Max-Content: sirve para que utilizen el espacio maximo/completo del objeto que contienen sin importar el tamaño

Minmax: minimo y maximo que puede medir el contenido

Min-content: es para que utilize el espacio minimo disponible, se usa dentro del repeat(minmax(min content(espacio minimo que puede usar), 200px(espacio maximo)))

Auto-Fill: Cuando se expande el contenedor, a medida que nos brinda espacio para mas items, los va insertando

Auto-fit: hace que los elementos que hay, se vayan expandiendo automaticamente para ocupar el espacio qeu tengan

La lineacion es General, por filas, columnas o individual. cada celda es un "flex-container", por eso se dice que la alineacion es individual.
para alinea la celda en el EJE X, se usa Justify-items: start, end o center; para alinear la celda en el eje Y se usa Align-items:start, center o end, el que viene por defecto es STRETCH; que diferencia hay entre Align-items y align-content, items, alinea las celdas propiamente dichas, y content, alinea la filas para alinear las alinear las columnas, se usa justify-content,

para que el 5 y 6 esten pegados como estan en el mismo eje X, se modifica el Justify:self, del 6 a start, o del 5 a END

para que el 4 y 7 esten juntos que estan en el EJE Y, se modifica el align-self del 4 con end y del 7 con start, asi uno sube y el otro baja

si se tiene que modificar el align y el justify, se puede cambiar en Place-self:align justify; como en el eleemnto 2

si quiero modificar la posicion de un elemento en el Todo, se usa Order.
